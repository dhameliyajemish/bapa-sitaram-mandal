# બાપા સીતારામ મંડળ મેનેજમેન્ટ સિસ્ટમ (Bapa Sitaram Mandal Management System)

આ એક આધુનિક MERN-stack આધારિત મંડળ મેનેજમેન્ટ સિસ્ટમ છે, જે સભ્યોની નોંધણી, માસિક બચત હપ્તો (hapto), ઉપાડ (upad), વ્યાજ (vyaj), દંડ (dand) અને લોનની વિગતો સરળતાથી રાખવા માટે બનાવવામાં આવી છે.

---

## 🚀 મુખ્ય વિશેષતાઓ (Key Features)

### ૧. સભ્યોનું સંચાલન (Member Management)
* સભ્યોની નોંધણી અને સંચાલન કરો.
* મહત્વની વિગતો સ્ટોર કરે છે: ખાતા નંબર (Fata No), સભ્યનું નામ, મોબાઇલ નંબર, ઈમેલ અને **શરૂઆતનું બેલેન્સ (Opening Balance)**.
* *નોંધ: સભ્યોના ફોર્મમાંથી સરનામાં (Address) નું ખાનું સંપૂર્ણપણે કાઢી નાખવામાં આવ્યું છે.*

### ૨. માસિક વ્યવહારો (Monthly Entry)
* **હપ્તો (Hapto / Collection)**: સભ્યોની માસિક બચત જમા રકમ.
* **ઉપાડ (Upad / Withdrawal)**: સભ્યો દ્વારા કરવામાં આવતો ઉપાડ.
* **વ્યાજ (Vyaj / Interest)**: ઉપાડ પર સભ્યો દ્વારા ચૂકવવામાં આવતું વ્યાજ.
* **દંડ (Dand / Penalty)**: માસિક મોડું જમા કરવા બદલ લાગતો દંડ.

### ૩. ઓટોમેટીક ૧% વ્યાજના નિયમો (1% Interest Rules)
* **બચત પર ૧% ચક્રવૃદ્ધિ વ્યાજ (Savings Compound Interest)**:
  * જમા રહેલ બચત સિલક પર દર મહિને **૧% ચક્રવૃદ્ધિ વ્યાજ (Compound Interest)** ગણાય છે.
  * આ ગણતરી સભ્યના **ઓપનિંગ બેલેન્સ (Opening Balance)** થી શરૂ થાય છે અને દર મહિને નીચેના સમીકરણ મુજબ ચાલે છે:
    $$\text{નવું બેલેન્સ} = \text{ગત મહિનાનું બેલેન્સ} \times ૧.૦૧ + (\text{હપ્તો} - \text{ઉપાડ} + \text{વ્યાજ} + \text{દંડ})$$
  * જો કોઈ મહિનાની એન્ટ્રી સિસ્ટમમાં કરવાની રહી ગઈ હોય, તો પણ સિસ્ટમ સમયગાળા મુજબ આપોઆપ દર મહિનાનું ૧% વ્યાજ ચડાવીને ગણતરી સાચી રાખે છે.
* **ઉપાડ પર ૧% વ્યાજ (Withdrawal Interest)**:
  * જ્યારે તમે એન્ટ્રી કરતી વખતે **ઉપાડ (Upad)** ના ખાનામાં કોઈ રકમ નાખશો, ત્યારે સિસ્ટમ આપોઆપ તેના **૧% વ્યાજ** ગણીને **વ્યાજ (Vyaj)** ના ખાનામાં મૂકી દેશે (ઉદા. ₹૫૦,૦૦૦ ઉપાડ કરવા પર ₹૫૦૦ વ્યાજ આપોઆપ આવી જશે).
  * એડમિન ઈચ્છે તો આ આપોઆપ ગણાયેલ વ્યાજમાં જરૂર મુજબ જાતે ફેરફાર (Edit) પણ કરી શકે છે.

### ૪. રીપોર્ટ અને પીડીએફ (Reports & PDFs)
* **માસિક રસીદ (PDF Receipt)**: દરેક સભ્ય માટે પ્રિન્ટ-રેડી પીડીએફ રસીદ ડાઉનલોડ કરી શકાય છે અથવા આખા મહિનાના બધા સભ્યોની રસીદ એકસાથે પણ પ્રિન્ટ કરી શકાય છે.
* **બેલેન્સ રીપોર્ટ (Ledger PDF)**: સંપૂર્ણ હિસાબ પત્રક જેમાં દરેક સભ્યની માસિક વ્યવહારોની વિગત અને છેલ્લી બચત સિલક દેખાશે.
* **વોટ્સએપ મેસેજ (WhatsApp Message)**: સભ્યોને તેમના માસિક વ્યવહાર અને બાકી બચત સિલકની વિગત સીધી વોટ્સએપ પર મોકલી શકાય છે.
* **ઈમેલ અહેવાલ (Email Statements)**: દરેક સભ્યને તેમના ખાતાની માસિક વિગતો સીધી જીમેલ (Gmail) દ્વારા મોકલી શકાય છે.

### ૫. સુરક્ષિત પાસવર્ડ રીકવરી (Password Recovery)
* નવીન રજીસ્ટ્રેશન (`/register`) અને લોગીન પેનલ.
* **Forgot Password**: એડમિન પોતાના રજીસ્ટર્ડ ઈમેલ પર OTP મેળવીને પાસવર્ડ રીસેટ કરી શકે છે (સુરક્ષા માટે OTP ક્યારેય સ્ક્રીન પર દેખાશે નહીં).

## 🛠️ વપરાયેલ ટેકનોલોજી (Tech Stack)

* **ફ્રન્ટએન્ડ**: React (Vite), Redux Toolkit, Bootstrap, React Icons, Chart.js
* **બેકએન્ડ**: Node.js, Express.js
* **ડેટાબેઝ**: MongoDB (Mongoose)

## ⚙️ સેટઅપ અને ચાલુ કરવાની રીત (Setup & Run)

### પૂર્વજરૂરિયાત (Prerequisites)
* [Node.js](https://nodejs.org/) (v16 કે તેથી નવું)
* [MongoDB](https://www.mongodb.com/) ચાલુ હોવું જોઈએ

### ૧. પ્રોજેક્ટ ડાઉનલોડ કરી ડિપેન્ડન્સી ઇન્સ્ટોલ કરવી
રૂટ ફોલ્ડર, સર્વર ફોલ્ડર અને ક્લાયન્ટ ફોલ્ડરમાં નીચેના કમાન્ડ ચલાવો:
```bash
# મેઈન ફોલ્ડરમાં
npm install

# સર્વર ફોલ્ડરમાં
cd server
npm install

# ક્લાયન્ટ ફોલ્ડરમાં
cd ../client
npm install
```

### ૨. એન્વાયરમેન્ટ વેરિયેબલ્સ સેટ કરવા (.env Setup)
`server` ફોલ્ડરની અંદર `.env` નામની ફાઇલ બનાવો અને નીચે મુજબની વિગતો લખો:
```env
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/mandalDB
JWT_SECRET=your_jwt_secret_key_here

# Gmail SMTP Configuration (ઈમેલ અને OTP માટે)
EMAIL_USER=your_gmail_address@gmail.com
EMAIL_PASS=your_google_app_password
```
> ⚠️ **મહત્વનું**: `EMAIL_PASS` માં તમારા જીમેલનો અસલી પાસવર્ડ નથી નાખવાનો, પણ તમારા ગૂગલ એકાઉન્ટના સેટિંગ્સમાંથી **App Password** જનરેટ કરીને તે નાખવાનો રહેશે.

## 🖥️ પ્રોજેક્ટ રન કરવો (Running the Application)

પ્રોજેક્ટના મેઇન (Root) ફોલ્ડરની અંદર નીચેનો કમાન્ડ ચલાવવાથી સર્વર અને ક્લાયન્ટ બંને એકસાથે ચાલુ થઈ જશે:

```bash
npm run dev
```

* **બેકએન્ડ (API)**: [http://localhost:5000](http://localhost:5000) પર ચાલુ થશે.
* **ફ્રન્ટએન્ડ (વેબસાઈટ)**: [http://localhost:5173](http://localhost:5173) પર ચાલુ થશે (જો આ પોર્ટ રોકાયેલ હોય તો `http://localhost:5174` પર).

## 📂 ફોલ્ડર સ્ટ્રક્ચર (Folder Structure)

```
├── client/              # રીએક્ટ ફ્રન્ટએન્ડ (Vite)
│   ├── src/
│   │   ├── api/         # Axios API કનેક્શન ફાઈલો
│   │   ├── context/     # સ્ટેટ મેનેજમેન્ટ Context
│   │   ├── pages/       # પેજીસ (MonthlyEntry, ReportsPage, Loans વગેરે)
│   │   ├── redux/       # Redux સ્લાઈસ
│   │   └── utils/       # ઉપયોગી હેલ્પર ફંક્શનો
│   └── package.json
│
├── server/              # નોડ/એક્સપ્રેસ બેકએન્ડ
│   ├── config/          # ડેટાબેઝ જોડાણ સેટિંગ્સ
│   ├── controllers/     # લોજિક કંટ્રોલર્સ (રિપોર્ટ અને એન્ટ્રી કંટ્રોલર)
│   ├── middleware/      # સિક્યોરિટી ચેક (JWT)
│   ├── models/          # MongoDB સ્કીમા (Member, MonthlyEntry, Admin)
│   ├── routes/          # Express API ના માર્ગો (Routes)
│   └── index.js         # બેકએન્ડ ચાલુ કરવાની મેઈન ફાઈલ
│
└── package.json         # મેઇન પ્રોજેક્ટ સેટિંગ્સ અને રન કરવાના કમાન્ડ્સ
```
